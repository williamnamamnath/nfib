# NFIB

This repository contains a React-based website in regards to neurofibromatosis (NFIB). The app is an informational site with pages for Home, NF1 background, team, news & events, past projects and contact information for the NFIB clinic at the Montreal Children's Hospital.

Status: Work in progress

- **Tech Stack**: Built with `react`, `javascript`, `CSS`, `bootstrap` and `styled-components`.
