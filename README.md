# NFIB

This repository contains a React-based website in regards to **neurofibromatosis** (NFIB). The app is an informational site with pages for Home, NF1 background, team, news & events, past projects and contact information for the NFIB clinic at the __Montreal Children's Hospital__.

- **Status**: `Work in progress`

- **Tech Stack**: Built with `react`, `javascript`, `CSS`, `bootstrap` and `styled-components`.
